## How to Deploy Online

### Option 1: Netlify (Easiest - Recommended)

1. Go to https://www.netlify.com
2. Sign up for free (or log in)
3. Drag and drop this entire folder onto the Netlify dashboard
4. You'll get a URL like `https://random-name-123.netlify.app`
5. Share the URL with your mom!

### Option 2: GitHub Pages

1. Create a GitHub account at https://github.com
2. Create a new repository
3. Upload all files to the repository
4. Go to Settings â†’ Pages
5. Select source: "Deploy from a branch"
6. Select branch: `main` and folder: `/ (root)`
7. Your site will be at `https://your-username.github.io/repository-name`

### Option 3: Vercel

1. Go to https://vercel.com
2. Sign up for free
3. Import your project (drag and drop or connect GitHub)
4. Deploy automatically
5. Get your URL instantly

## Local Testing

To test locally, simply open `index.html` in your web browser.

## Notes

- The website uses microphone access for the candle blowing feature
- Make sure to allow microphone permissions when prompted
- Works best on modern browsers (Chrome, Firefox, Safari, Edge)
